CodeNotation

Hello, this is a python code editor that allows you to express your thinking processes. During your programming, you can select any text and hit an annotation option that best reflects your thought frequently.
Ideally, text -> what you are focusing and annotation -> what you are thinking.
We may remind you to annotate if you haven't done so for a while. You don't need to worry about submitting the answer for now.
Please let us know your opinions about this application. Thank you!
